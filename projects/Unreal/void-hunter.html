Void Hunter

Modular Gameplay Systems & Data-Driven Wave Architectures

Engine Unreal Engine 5.7
Genre Melee Action / Tactical Waves
Role Gameplay Systems Engineer
Timeline March 1, 2026 - Present

Project Overview

Void Hunter is a high-octane melee combat game built from the ground up in Unreal Engine 5.7. While sourcing high-quality assets for visuals and audio, I was responsible for the entirety of the project's C++ architecture, programming all core systems and seamlessly integrating animations, sound effects, and VFX with custom code. The project highlights modular, designer-friendly systems that prioritize performance and visceral feedback.

Key Features

Data-Driven Wave System: Architected a robust C++ Wave Manager that orchestrates enemy encounters using custom FWave structs and robust FTimerHandle state management.
Modular Component Architecture: Engineered decoupled Health and Combat components to manage character states, melee hit reactions, and damage processing.
Custom AI State Logic: Developed a high-performance C++ AI system for Melee and Ranged enemies with tactical movement and centralized "Think" loops.
Full System Integration: Successfully integrated external animation montages, spatialized audio, and Niagara VFX into C++ logic for a cohesive gameplay experience.
Combat Feedback: Implemented Frame-Freeze (Hit Stop) and dynamic Camera Shakes within the Combat Component for visceral physical feedback.

Technical Responsibilities

Resolved complex race conditions in wave transitions by implementing robust state guards and timer management.
Built the core melee combat loop from scratch, prioritizing performance and clean code principles (DRY, SOLID).
Utilized Unreal Engine Interfaces to ensure seamless communication between unrelated actors (Player, Enemy, Projectiles).
Configured custom NavMesh bounds and AI movement parameters to ensure fluid enemy traversal in complex layouts.
Exposed core parameters to the Editor, allowing for rapid iteration on wave difficulty and enemy tuning.
